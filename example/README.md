# Universal Viewer Example

This example demonstrates how to use the Universal Viewer package.

## Features Demonstrated

- Loading content from URLs (websites, PDFs, videos, etc.)
- Loading files from device
- Custom themes
- Callbacks for lifecycle events
- Different content types

## Running the Example

### Web

```bash
flutter run -d chrome
```

## Examples in the App

- DuckDuckGo - Display any website
- PDF Sample - View PDF documents
- YouTube - Embed YouTube videos
- Image - Display images
- File Picker - Load local files

## Customization

### The example shows how to:
- Configure themes
- Handle callbacks 
- Enable/disable features 
- Handle errors
